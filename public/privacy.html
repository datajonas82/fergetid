Personvernerklæring

Sist oppdatert: 2025-08-27

FergeTid behandler personopplysninger i tråd med gjeldende regelverk. Denne siden beskriver hvilke data vi behandler, hvorfor og hvordan du kan kontakte oss.

Hvilke data samles inn

  • Posisjonsdata for å finne fergekaier i nærheten (valgfritt).
  • Abonnementsstatus via våre betalings- og abonnementspartnere.
  • Tekniske metrikk for feilretting og forbedring.

Formål

Data brukes for å levere kjernefunksjoner (f.eks. nærmeste fergekai, kjøretider) og for å administrere Premium-abonnement.

Lagring og deling

Data deles kun med nødvendige tredjepartsleverandører (f.eks. kart- og betalingsleverandører) for å levere tjenesten.

Dine rettigheter

Du kan be om innsyn, retting eller sletting der det er relevant. Kontakt oss på support@fergetid.app.
